- Math is a school subject that  is problematic for most students - That's why matmaplus was created!

## Technologies Used
- Java 17
- Spring
- Spring boot
- MySQL
- Bootstrap (HTML,CSS)
- Thymleaf


## Features
List the ready features here:
- Login
- Registration
- Adding courses to cart
- Comment and rating courses
- View purchased courses
